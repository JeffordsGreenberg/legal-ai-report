# Legal AI Tool Review

An independent, practitioner-scored comparison of AI tools for legal teams.

**Live site:** [legal-ai-report.laurajg.com](https://legal-ai-report.laurajg.com)

## What this is

Every legal AI comparison I could find was written by analysts who hadn't used the tools, 
or sponsored by the vendors being reviewed. This is the alternative — scored by working 
lawyers, with full disclosure of relationships and conflicts.

## Stack

- **Next.js / React** — frontend
- **Vercel** — hosting
- **Notion** — reviewer submission database
- **Anthropic API** — live AI news feed per tool
- **GitHub** — version control

## How it was built

Built by a GC with no engineering background using Claude to write all the code. 
No developer. No budget. One day.
